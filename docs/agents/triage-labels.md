# Triage labels

This repository uses canonical triage labels. Skills that run triage should apply these exact strings.

## Canonical mapping

- `needs-triage`: maintainer evaluation required
- `needs-info`: waiting on reporter clarification
- `ready-for-agent`: fully specified and AFK-agent ready
- `ready-for-human`: requires human implementation
- `wontfix`: intentionally not actioned

## Guidance

- Do not create alternate synonyms unless this mapping is intentionally changed.
- If labels are renamed in GitHub, update this file and `AGENTS.md` together.
